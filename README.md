# Mindustry Tea Mod ☕

Ultra rýchle stavanie s čajom!

## Obsah
- Čajová ruda
- Liquid Tea (5x stavanie)
- Black Tea (15x stavanie)
- Tea Sprayer buff block

## Inštalácia
1. Stiahni mod
2. Daj do `Mindustry/mods/`
3. Aktivuj v hre

## Použitie
Postav Tea Sprayer, pripoj čaj, stoj v dosahu 80 blokov = ultra rýchle stavanie!
